Exploratory Data Analysis (EDA) - Marketing Campaign Dataset

Overview

This repository contains my Exploratory Data Analysis (EDA) project completed as part of the SkillCraft Technology Internship.

Objective

The objective of this project is to analyze the marketing campaign dataset, identify patterns and trends, and derive meaningful business insights using Python.

Tools & Libraries

- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Jupyter Notebook

Project Workflow

- Data Loading
- Data Cleaning
- Data Exploration
- Statistical Summary
- Data Visualization
- Insights and Conclusions

Key Insights

- Analyzed customer demographics and purchasing behavior.
- Identified relationships between income, age, and spending.
- Visualized data distributions and correlations.
- Detected patterns that can support business decision-making.
